Introduction to Visual Studio Code


What is Visual Studio and how to set it up?


Visual Studio is an IDE developed by Microsoft used for software development.


To set-up Visual Studio, follow the steps:

Download Visual Studio.

Open the IDE.

Create a folder on your computer.

Click on the 'EXPLORER' Icon.

Click on 'File' and select 'Open Folder'.

A dialog box opens. Select the folder from your desktop.

Your new project will open.

In the 'Explorer' Section, Click on the New File Button.

You can create files like 'index.html', 'style.css', etc.

You can download extensions from the 'EXTENSIONS' Section.



What is the Console and why is it used?


The console object provides access to the browser's debugging console. It is useful for testing purposes on your Webpage.



How to open the Console Screen and print anything on the console?


In the Visual Studio Program, click on the "Terminal" Option in the 'VIEW' Tab or use the Ctrl+` shortcut.

In the Webpage, left-click and select the 'Inspect' Option. Select the 'Console' option to view the Console Screen.


Use the console.log() method to print anything on the console.
